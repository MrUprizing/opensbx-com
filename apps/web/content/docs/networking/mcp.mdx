Opensbx includes an MCP-compatible endpoint at:

```text
/v1/mcp
```

## Deployment Note

When running with `-base-domain your-domain.com`, MCP works without `MCPGODEBUG`.

## Initialization Check

```bash
curl -i https://your-domain.com/v1/mcp \
  -X POST \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"curl","version":"1.0"}}}'
```

Use this as a quick smoke test after deploying your tunnel, domain, and API endpoint.
